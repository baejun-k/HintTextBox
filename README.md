# TextBoxWithHint  
  
WPF의 TextBox에 Hint 기능이 없어서 사용할 수 있게 간단하게만 만들었다.  
TextBox를 상속하였음.
  
## 추가한 속성  
* Hint  
TextBox의 Text가 비었을 때 보이는 글자  
* HintForeground  
Hint의 글자 색
* HintBackground  
Hint가 보일 때의 배경색. null일 때(미설정) 기본 TextBox의 배경색을 따라감.
